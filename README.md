# Twiputout

## アプリケーション概要
- さっと簡単に記録できるメモアプリです。
- SNSのタイムラインのように、縦一列の時系列でメモを見返すことができます。
- これは特に大事、覚えておきたい、というメモはお気に入り登録して別途一覧で閲覧可能です。

## 利用方法
名前、メールアドレス、パスワードで簡単に登録できます。
登録後でもヘッダーの登録したユーザーネームのプルダウンからマイページで変更可能です。
投稿するボタンをクリックしてテキスト、画像をメモとして投稿できます。
投稿したメモはeditボタンから編集でき、deleteボタンから削除できます。
別途一覧で残しておきたいメモがあれば、☆マークをクリックするとお気に入り登録ができ、★に変わります。
ヘッダーの登録したユーザーネームのプルダウンから、お気に入りをクリックすると、
お気に入り登録したメモの一覧をみることができます。
Twiputoutのアプリ名をクリックするとメモ一覧に遷移します。

## 目指した課題解決
学生や社会人の方が、備忘録として残したい言葉、スケジュール、日常の中でふと何か思いついたアイディア、
好きな本やお店など、書き留めておきたいことがあった時にすぐに書き留められるようシンプルな構造にしました。
フォルダを探して書く手間、探す手間を省きたい、フォルダ作成の工数も割愛したい、という点を解消し、
できる限り選択する回数を減らして簡単に使えるよう、投稿ボタンのみにし、時系列がわかりやすいように最新のものから表示、
縦一覧にすることによってスクロールだけで簡単に見返すことができる、という仕様にしました。

### ユーザー管理機能
- 機能
ユーザー管理機能

- 目的
使用するユーザーの情報を管理する
ログインして自分のマイページをみることができる

- 詳細
ユーザーネーム、メールアドレス、パスワードを入力してアカウント登録する。

ログイン後、ヘッダーにある登録したユーザーネームのプルダウンのマイページから自分の登録情報が表示されている。

新規登録できたらヘッダーのボタンが登録したユーザーネームと、投稿するボタンが表示される。
登録したユーザーネームのプルダウンからマイページを押すとマイページに遷移し、ログアウトを押すとログアウトできる

# DB設計

## user table
| Column      | Type   | Options     |
|-------------|--------|-------------|
| user_name   | string | null:false  |
| password    | string | null:false  |
| email       | string | unique:true |

## Association
- has_many :memos
- has_many :fav_memos


## memo table
| Column     | Type       | Options           |
|------------|------------|-------------------|
| memo_text  | text       | null:false        |
| user       | references | foreign_key: true |

## Association
- belongs_to :user
- has_many :fav_memos


## fav_memo table
| Column | Type       | Options           |
|--------|------------|-------------------|
| user   | references | foreign_key: true |
| memo   | references | foreign_key: true |

## Association
- belongs_to :user
- belongs_to :memo
